# Brewski Bets

Source code for [brewskibets.com](https://brewskibets.com) — a web app for making friendly beer bets between friends.

## Stack

- **Frontend**: Angular, SCSS, `@eagami/ui`, Clerk authentication
- **Backend**: Hono, Drizzle ORM, PostgreSQL (Neon), Cloudflare R2
- **Deployment**: Vercel (frontend SPA + serverless API functions)
